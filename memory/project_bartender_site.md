Сайт бармена и изобретателя коктейлей **Vahan Asoyan** расположен в `dev4/index.html` — единый HTML файл без внешних зависимостей.

**Why:** Пользователь попросил создать персональный сайт бармена с рецептами классических коктейлей, минималистичным современным дизайном.

**How to apply:** При любых изменениях дизайна, текста или структуры — редактировать именно `dev4/index.html`. Обновлять эту память при каждом изменении.

## Текущее состояние (2026-05-16)

### Дизайн
- Цветовая схема: **чёрно-белая**
  - Фон: `#000000`
  - Поверхность: `#0c0c0c`
  - Граница: `#2a2a2a`
  - Текст: `#f2f2f2`
  - Приглушённый: `#777777`
  - Акцент: `#ffffff` (белый)
- Шрифты: **Cormorant Garamond** (заголовки, serif) + **Inter** (текст, sans-serif)
- Картинки: **цветные** (без grayscale фильтра), с фильтром `brightness`
- Стиль: минималистичный, тёмный, современный

### Структура страницы
1. **Nav** — фиксированная, логотип `V.A.`, blur-эффект
2. **Hero** — имя Vahan / Asoyan, фото бармена за стойкой
3. **Gallery Strip** — 5 атмосферных фото бара, анимация расширения при hover
4. **About** — биография, 3 фото, статистика (10+ лет, 80+ рецептов, 12 наград)
5. **Cocktails** — сетка 3×2, 6 классических коктейлей с рецептами при hover:
   - Negroni, Old Fashioned, Margarita, Espresso Martini, Mojito, Dry Martini
6. **Signature** — авторский коктейль «Золотой туман» с пошаговым рецептом
7. **Footer** — имя Vahan Asoyan, ссылки
