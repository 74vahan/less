После каждого изменения в проекте — обновлять соответствующий memory-файл, чтобы он отражал актуальное состояние.

**Why:** Пользователь явно попросил хранить память в файлах внутри проекта и обновлять её при каждом изменении.

**How to apply:** При любом изменении `dev4/index.html` (цвет, имя, структура, контент) — сразу же обновлять `memory/project_bartender_site.md`, отражая новое состояние.
